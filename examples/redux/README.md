## Redux Demo

This component generate the configuration from a smart component connected to redux.

It will also dispatch a redux action on success.

## Run the demo

Install:

```bash
npm install
```

Run

```bash
npm start
```

Open your browser at [http://localhost:8080/](http://localhost:8080/).
